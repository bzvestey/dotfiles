# Hyprland Keybindings

This document lists the configured keybindings for Hyprland in this NixOS configuration.
The `SUPER` key is the `Windows` key (or `Command` key on some keyboards).

## General
| Keybinding | Action |
| :--- | :--- |
| `SUPER + T` | Open Terminal (Ghostty) |
| `SUPER + E` | Open File Manager (Nautilus) |
| `SUPER + SPACE` | Open App Launcher (Wofi) |
| `SUPER + Q` | Close Active Window |
| `SUPER + V` | Toggle Floating Window |
| `SUPER + M` | Exit Hyprland |
| `SUPER + P` | Pseudo Tiling (Dwindle) |
| `SUPER + J` | Toggle Split (Dwindle) |

## Window Focus
| Keybinding | Action |
| :--- | :--- |
| `SUPER + ←` | Move focus Left |
| `SUPER + →` | Move focus Right |
| `SUPER + ↑` | Move focus Up |
| `SUPER + ↓` | Move focus Down |

## Workspaces
| Keybinding | Action |
| :--- | :--- |
| `SUPER + [0-9]` | Switch to Workspace [0-9] |
| `SUPER + Scroll Up` | Switch to Previous Workspace |
| `SUPER + Scroll Down` | Switch to Next Workspace |

## Move Windows
| Keybinding | Action |
| :--- | :--- |
| `SUPER + SHIFT + [0-9]` | Move active window to Workspace [0-9] |
| `SUPER + LMB (Drag)` | Move Window |
| `SUPER + RMB (Drag)` | Resize Window |

## Multimedia & Brightness
| Keybinding | Action |
| :--- | :--- |
| `Volume Up` | Increase Volume (5%) |
| `Volume Down` | Decrease Volume (5%) |
| `Mute` | Toggle Audio Mute |
| `Mic Mute` | Toggle Microphone Mute |
| `Brightness Up` | Increase Brightness (10%) |
| `Brightness Down` | Decrease Brightness (10%) |
